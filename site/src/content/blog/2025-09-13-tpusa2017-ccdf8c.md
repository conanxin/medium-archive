### 黑金入侵学生会？TPUSA被指"种族歧视+非法竞选"（2017） {#49a5 .graf .graf--h3 .graf--leading .graf--title name="49a5"}

<figure id="c96d" class="graf graf--figure graf-after--h3">
<img src="https://cdn-images-1.medium.com/max/800/0*p_jEX_dzxaoPhpiy" class="graf-image" data-image-id="0*p_jEX_dzxaoPhpiy" data-width="1200" data-height="800" data-is-featured="true" />
</figure>

周二，在佛罗里达州西棕榈滩的一处会展中心，在"USA!"和"边境墙将会建成！"的呼喊声中，小唐纳德·特朗普为 Turning Point USA（一家保守派非营利组织）举办的为期三天的年度峰会拉开序幕。Turning Point 总部设在芝加哥郊外，其目标是在美国大学校园引发一场政治革命，部分方式是向全美各地的学生会选举输送资金，以让右翼倾向的候选人当选。但该组织对其资金及捐助者讳莫如深，这就带来了这样一种可能性：所谓"黑金"如今或许不只在影响州和联邦层面的选战，还在塑造校园选举。

Turning Point 大肆宣扬其与总统家族的密切关系。该组织的网站连周预热小唐纳德的出场，配有他举起握拳的照片。其宣传材料还引用了这位小特朗普的赞语："你们所做的事""简直令人惊叹。"小唐纳德的兄弟埃里克的妻子拉拉·特朗普也参与了该组织。周三在西棕榈滩，她主持了一场午餐会，为 Turning Point 即将举行的"青年女性领导力峰会"造势。该组织 24 岁的执行主任兼创始人查理·科克告诉我，他把小唐纳德视为"一位私人朋友"。

Turning Point 将自己塑造为对其所认为的大学校园里自由派不宽容现象的草根回应。科克称大学校园是"极权主义的孤岛"；他与其支持者坚称，保守派才是美国真正的歧视受害者，他誓言要代表他所称的"右翼战队"进行反击。科克是福克斯新闻的常客，去年夏天他受邀在共和党全国代表大会上发表演讲。科克说，他就是在那里结识了小唐纳德，并与之"一拍即合"。大会结束后，科克把时间一分为二，一方面投入 Turning Point 的活动，另一方面作为青年外联专员为特朗普竞选团队工作。科克对我说（指小唐纳德）："我帮他一起协调了一些相当成功的活动"，"我也帮他提过包。"今年早些时候，朋友们为科克办了一场惊喜生日派对，小唐纳德到场，前特朗普白宫顾问塞巴斯蒂安·戈尔卡也出席了。

随着 Turning Point 的声势上升，外界对其资金来源和策略的审视也随之加剧。我获得的内部文件以及对前员工的采访表明，该组织可能规避了禁止慈善组织参与政治活动的竞选资金法律。前员工称，他们被指示与知名保守派人士合作，其中包括最高法院大法官克拉伦斯·托马斯的妻子，以在 2016 年为共和党总统候选人提供协助。或许对一个宣称保守派才是美国歧视真正受害者的组织而言，更令人不安的是，Turning Point USA 还被指涉嫌营造了一种对少数族裔敌视的氛围。一位消息来源提供给我的截图显示，克里斯托·克兰顿（直到去年夏天一直担任该组织的全国外勤总监）曾给另一名 Turning Point 员工发短信说："我恨黑人。就他妈的让他们全都滚蛋......我恨黑人。就这样。"

在担任该组织第二号人物五年后辞职的克兰顿起初拒绝置评。她通过电子邮件对我表示："我已不在 Turning Point 工作，也不希望成为报道的一部分。"随后在第二封电子邮件中，她说："我对这些信息毫无记忆，它们并不反映我的信念或我是谁，而且在我还是青少年时也是一样。"

收到克兰顿短信的前 Turning Point 员工约翰·瑞安·奥罗克要求"不要在任何有关 Turning Point 的文章或背景信息中使用这些消息"，并拒绝就此发表评论。科克在一封电子邮件中表示，"Turning Point 在获悉问题后的 72 小时内评估了情况并采取了果断行动。"不久之后，克兰顿离开了该组织。

据前员工称，尽管科克作为 Turning Point 的公众代言人对外发声，但其前外勤总监克兰顿则充当了具体操盘的负责人。在科克与布伦特·哈马切克于 2016 年合著的《转折点之时：为后代设定通往自由市场与有限政府之路》一书中，他称克兰顿是"我们所能做出的最佳雇用"。他称她"在事实上担任首席运营官期间，对 Turning Point 的成功不可或缺"。他还补充道："Turning Point 需要更多的克里斯托；美国也是如此。"

Turning Point 的前员工表示，该组织是个难以工作的地方，充斥着紧张气氛，其中一部分与种族有关。三年前在该组织工作时担任外勤总监的前员工加布里埃尔·费基耶尔对我说，她是唯一被聘为外勤总监的非裔美国人。她说："回头看，我觉得这就是种族主义。当时我在责怪自己，以为是我做错了什么。"现为模特的费基耶尔回忆说，她招入组织的年轻黑人成员突然发现自己被取消了参加该组织年度学生峰会的邀请，而当她本人出席时，她看到台上的一些演讲者"恶语中伤黑人女性未婚生下这么多孩子。那真的很冒犯。"（科克通过发言人否认发生过任何此类事件，并表示："这些指控完全毫无根据，甚至荒谬可笑。"）

费基耶尔说，克兰顿在马丁·路德·金日以她工作表现不佳为由解雇了她。费基耶尔告诉我："我是他们唯一的黑人美国员工，而他们竟然在 M.L.K. Day 把我解雇 --- --- 这也太无礼了！"她还表示，"因为我是黑人，我在那里工作感到非常不舒服"，但她也说她也见到白人员工遭受不当对待。"我的民主党朋友曾告诉我，有些共和党人不关心穷人和少数族裔，我原以为这不是真的，但后来我找到了他们所说的那些人！"

在各大学校园举办的 Turning Point 活动上，演讲者被指不遗余力地对种族与文化敏感性嗤之以鼻。比如，保守派挑衅者米洛·扬诺普洛斯（Milo Yiannopoulos） --- --- 他的出场由 Turning Point 与科罗拉多大学博尔德分校的大学共和党人（College Republicans）共同主办 --- --- 曾表示，尽管自己是同性恋，他憎恨"基佬"、女同性恋者和女权主义者；他说，这些人"他妈的就是恨男人"。

为嘲弄校园对仇恨言论的反对，肯特州立大学的 Turning Point 分会成员在去年秋天发起了一场抗议，他们穿着成人纸尿裤、叼着安抚奶嘴出现在校园里，同时宣称"安全空间是给孩子的"。这场抗议引发了广泛嘲笑，科克的发言人表示他不赞成这种做法，并随后发布了指导方针，反对其他分会重复此举。

科克在伊利诺伊州惠灵长大，是一名鹰级童子军；他在 2015 年对硅谷保守派论坛的演讲中表示，他"一生中头号梦想"就是就读西点军校，但他认为属于自己的名额却被"性别不同、立场不同且资历差得多的候选人"拿走了，他声称自己知道那人的考试分数。（科克表示，他发表这番言论时是在讽刺。）一位年长的相识鼓励他放弃上大学，创办一个保守派版本的进步派倡议组织 MoveOn.org。科克在一次采访中承认，他本人从未真正上过大学，却领导一个致力于在校园开展政治斗争的组织，这多少有些讽刺。科克对我说："我打趣说，我不够聪明，没法上四年制学校，"不过他指出，自己随后在一所社区大学继续了学业。

然而，MoveOn 的一部分被设为超级政治行动委员会（super PAC），另一部分为 501(c)4"社会福利组织"，二者在法律上都被允许参与政治选举。它还有一项政策：披露任何捐款达到五千美元或以上者的姓名。相比之下，Turning Point 是一家 501(c)3 慈善机构。这意味着，与 MoveOn 的捐助者不同，Turning Point 的捐助者可以就其捐款享受税收抵扣并保持匿名。但作为这些好处的交换，国税局（IRS）严格禁止 Turning Point 之类的慈善机构直接或间接参与政治选举。

多位 Turning Point 的前员工在采访中告诉我，他们觉得自己被要求参与的活动越过了竞选资金法律为此类团体划定的界限。在 2016 年总统竞选期间为 Turning Point 工作的佩登·霍尔告诉我，她的上司克兰顿曾发邮件到她的 Turning Point 工作邮箱，安排她与最高法院大法官克拉伦斯·托马斯的妻子金妮·托马斯进行协调，以协助泰德·克鲁兹的总统竞选。霍尔回忆道："模糊之处就是从那时开始的。"她说，不久之后，支持克鲁兹参选并担任 Turning Point 咨询委员会成员的金妮·托马斯给霍尔和她的妹妹（也在 Turning Point 工作）留了语音消息，称她将寄去两百块克鲁兹的标牌，供她们在即将到来的威斯康星州总统初选中分发。

"克里斯托在未获我许可的情况下把我的私人邮寄地址给了金妮·托马斯，"霍尔回忆道。"他们把员工的个人信息提供给了一位最高法院大法官的妻子。"她说，接下来她就发现，数百块克鲁兹的标牌寄到了她家。"我们把它们扔了，"霍尔说。她本人是克鲁兹的支持者，但她表示，"我们想按照我们自己的方式去做志愿者，而不是屈从于上司的压力。我觉得即便这没有越过法律红线，也越过了职业界线。"

联邦选举委员会的前共和党委员、无党派竞选资金法监督组织"竞选法律中心"的创始人兼主席特雷弗·波特表示，Turning Point 被禁止协助政治竞选活动。他说："依据法律，501(c)(3) 不能从事政治行动，也不能向竞选活动提供任何有价值的东西，包括学生，或学生的姓名。如果 Turning Point USA 所做的是在校园里帮助共和党人，并把他们输送给竞选团队，那就是一项政治行动，听起来似乎已越线。"

通过电话联系时，金妮·托马斯拒绝置评。克兰顿的律师罗伯特·格拉伯曼表示，如果她"是在霍尔的 TPUSA 邮箱地址发了电子邮件，那只是克兰顿女士的一个无心疏忽与真诚的错误。克兰顿女士断然否认使用 TPUSA 资源来协助任何政治竞选活动。她完全理解 501(c)(3) 指南，并在许多情况下咨询过法律顾问，以确保所有个人参与的竞选活动都符合 501(c)(3) 规则。"

苏珊·沃克于 2016 年在佛罗里达州为 Turning Point USA 工作，她告诉我，该组织确实援助了共和党的政治竞选。沃克说，她在 Turning Point 工作期间所创建的一份包含数百名学生支持者姓名的名单，在她不知情的情况下被交给了一名为马可·鲁比奥总统竞选工作的人员。她说："那份名单大概有七百名学生，是我拼命干出来的。我给每个我谈过话的学生都加了备注，而这些备注当时都还在上面。"她补充说，那名鲁比奥团队的干事"本不该拿到那份名单。我们是慈善机构，而他是在政治竞选团队。"

来自南卡罗来纳州和俄亥俄州的其他 Turning Point 前员工的电子邮件和采访也显示，总统竞选工作与该慈善机构的工作存在混同。在南卡罗来纳州，一串电子邮件显示，应两名支持克鲁兹的超级政治行动委员会官员的要求，科克让一名 Turning Point USA 员工召集学生支持克鲁兹。在 2016 年 1 月 25 日的一封电子邮件中，德鲁·里恩（Turning Point 咨询委员会成员，彼时协助运营一家支持克鲁兹的超级政治行动委员会）要求科克让另一名 Turning Point 员工向该超级政治行动委员会"送来尽可能多的学生"。里恩曾任共和党全国委员会副主任，他解释说，他需要"尽可能多的学生，用于一篇关于南卡罗来纳州相关行动的 WSJ 报道"。电子邮件线程显示，在科克同意提供帮助后，他与丹·特里普进行协调；特里普是里恩在该支持克鲁兹的超级政治行动委员会的同事，负责其在南卡罗来纳州的运营，并且是 Ground Game Strategies 的创始人兼总裁。

"好！"当特里普向 Turning Point 请求帮助时，科克答道。"南卡罗来纳州的哪个地区？"

"格林维尔、斯帕坦堡或安德森县，"特里普回复。

"一天中的什么时间？持续多久？"科克问。

"我在想周日下午晚些时候，两小时。拉票、培训，还有披萨，"特里普回应。

"明白了，很快去勘察，"科克在邮件中回复。科克解释说，南卡罗来纳州一名叫安娜·斯科特·马什的 Turning Point 员工将担任负责人，并补充道："安娜会来协助。让我们干起来！"

不久之后，电子邮件显示，这名 Turning Point 员工马什承诺召集所需的招募对象。"今晚会发出一些通知，希望明天给你一份名单......我确信我们在这里能找到一些可靠的学生。"马什拒绝就她的邮件置评。

当被问及这些做法时，科克让我参阅其律师莎莉·瓦根梅克的声明："Turning Point USA 一直勤勉地完全遵守所有规范非营利组织的相关法律法规。Turning Point USA 专注于财政保守主义、自由市场经济以及相关的学生教育与倡议，所有活动完全处于适用的 501(c)(3) 法律约束之内。"

里恩证实了上述交流确有其事，但称科克是"通过他的个人邮箱并在个人时间！"给他发的邮件。特里普也证实了这些邮件，但表示："我们欢迎许多志愿者参与我们的工作，并对他们的支持心怀感激。如果竞选资金规则被解读为阻止保守派志愿者行使参与政治进程的权利，那将相当令人不安。"

在一次电话采访中，科克拒绝透露为其组织提供超过八百万美元年度预算的捐助者姓名，并指出许多人更愿意保持匿名。但科克曾在多场能源行业的闭门会议上演讲并募款，其中包括 2017 年全美矿业协会董事会会议和 2016 年美国独立石油协会年会。在我们的采访中，科克承认，他的一些捐助者"在化石燃料领域"。

科克与化石燃料大亨的联系之所以引发争议，是因为 Turning Point 曾在校园内协助组织反对那些呼吁学校从化石燃料公司撤资的学生。Turning Point 向大学生分发了一本由科克作序的小册子，题为《化石燃料改善我们日常生活的 10 种方式》。在其中，他论称："在全国范围内，大学生正 clamoring 要求他们的校园从化石燃料中撤资......学生被灌输去相信这样一个神话：化石燃料很肮脏，而可再生能源是一个可行的替代方案......"Turning Point 还运营一个线上"教授观察名单"（Professor Watch List），针对其认为是自由派的教授；该小册子将这些"神话"的"延续"归咎于"左翼教授"。在采访中，科克对我说："我们认为将矛头对准化石燃料相当不公平，而让大学偏袒某一种政治议程而非另一种，其实并不符合大学的最佳利益。"他说，在能源行业会议上发表这一观点时，"反响很好"。

去年五月，《高等教育纪事报》发表了一篇关于其所谓 Turning Point "隐形政治影响计划"的调查报道。报道回顾了多个校园上的指控，称该组织向学生会选举输送资金，违反了学校设定的支出上限和透明度要求。报道详述了在马里兰大学和俄亥俄州立大学，得到 Turning Point 支持的学生候选人如何在"被抓到违反支出规则并试图隐瞒他们从 Turning Point 获得的帮助之后"被迫退出校园选举。报道还援引科克在 2015 年一次在保守派政治团体面前的露面时的话称，他的组织在学生会选举上"投入了大量时间、金钱和精力"。（在该报道中，科克否认有任何不当行为，并表示所谓存在某种秘密计划"完全荒唐可笑"。）

我获得的一份为潜在捐助者准备的 Turning Point 宣传册副本，让人得以一窥该组织的策略。（一名 Turning Point 前员工称，该宣传册被严密保管，未在网上发布，以防泄露。）其中的"校园胜利项目"被描述为一项详细的、多阶段的计划，旨在"夺取各所最知名、最具影响力的美国大学中学生会主席这一最高职位"。

第一阶段要求在"Power 5"联盟学校取胜，其中包括大西洋海岸联盟、十大联盟、太平洋12校联盟、十二大联盟和东南联盟。第二阶段要求在每一所 NCAA 一级学校赢得学生政府的最高职位，据称这些学校超过三百所。宣传册称，在该计划的前三年里，该组织的目标是在其中八成学校赢得学生政府职位的"绝对多数"。

一旦控制了学生政府，宣传册称，Turning Point 预期其盟友的校园领袖将遵循一套既定的政治议程。其中的要点包括削减对校园进步派组织的资助，实施"言论自由"政策以消除对仇恨言论的障碍，以及阻止所有校园"抵制、撤资与制裁"运动。Turning Point 的议程还呼吁其赋权的学生领袖动用学生资源举办演讲与论坛，在校园内推广"美国例外论与自由市场理念"。

如今，Turning Point 声称其在全美一千多所大学校园有布局，并拥有"比所有左翼校园团体加起来更强大、更有组织的存在"。科克告诉我，他的组织在过去一年新成立了三百个分会。《校园胜利项目》宣传册点名了五十多所四年制学院与大学，称该组织在 2016--17 年度帮助促成了这些学校学生政府的胜选，其中包括加州大学洛杉矶分校、锡拉丘兹大学、普渡大学、密歇根州立大学、维克森林大学和南加州大学；它还点名了另外一百二十二所学校，称该组织希望在第二阶段"夺取"其学生政府。宣传册指出，完成这一任务需要资金：具体为 220 万美元。

科克在采访中否认这些资金会直接用于学生的竞选活动。他说："我们不会直接资助任何这些候选人。"相反，他解释道，"我们会通过不同层级的领导力来支持他们"，包括培训以及他所称的"领导力奖学金"。

"黑金"（即匿名捐助者对全国性意识形态团体的捐款）流入校园选举的前景让一些学生感到不安。去年担任俄亥俄州立大学学生政府副主席的 Danielle Di Scala 说："学生们对我们的选举受到外部影响感到愤怒。我以前从未见过这种情况，但它正在成为一种趋势。"她对我说，"问题在于，当其他人从资金无限的团体那里获得资助时，这会把一些学生候选人挤出'市场'。"

全国校园领导力委员会执行主任 Andy MacCracken 表示，他担心校园选举对外来资金"尤其脆弱"，"因为实际上并没有任何标准规则"。MacCracken 说，"令人震惊的是，Turning Point 的运作规模之大"，并补充说，"据我所知，左翼团体并没有什么可以与之相提并论的"。他认为，这种推动反映了保守派对学生领袖未来价值的认识。"我完全可以想象他们在想，如果我们能在校园里赢下这一局，他们将来就会成为思想领袖。这是一种在一开始就高效取胜的方式。不过挑战在于，"他说，"其中有太多是不透明的。"
